# 🌐 Flutter Web

> Exercises and projects in Flutter Web, covering routes, file structure, file upload, landing pages, and responsive design.

This repository was created for **learning purposes**. It contains three independent Flutter Web projects, each focused on a different set of real-world concepts — from app structure and navigation to UI design and admin dashboards.

---

## 📁 Project Structure

```
Flutter_web/
├── bases_web/
│   ├── router/           # Route configuration
│   ├── ui/               # UI components and screens
│   └── main.dart
│
├── admin_dashboard/
│   ├── providers/        # State management
│   ├── router/           # Navigation and routes
│   ├── services/         # Business logic and file upload
│   ├── ui/               # Screens and widgets
│   ├── locator.dart      # Dependency locator
│   └── main.dart
│
└── vertical_landing_page/
    ├── providers/        # Scroll and state providers
    ├── router/           # Route setup
    ├── ui/               # Landing page sections
    └── main.dart
```

---

## 🗂️ Projects

| Project | Description |
|---------|-------------|
| **bases_web** | Flutter Web foundations — routing, file structure, and core UI patterns |
| **admin_dashboard** | Admin panel with login, registration forms, file upload, and services layer |
| **vertical_landing_page** | Scrollable landing page with responsive sections and providers |

## 🚀 Getting Started

1. Install [Flutter](https://flutter.dev/docs/get-started/install) with Web support enabled
2. Clone the repository
   ```bash
   git clone https://github.com/AsllyZuniga/Flutter_web.git
   ```
3. Navigate to any project and run it
   ```bash
   cd admin_dashboard
   flutter run -d chrome
   ```
